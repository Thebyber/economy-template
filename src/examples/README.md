# Examples

Optional reference implementations — **not** wired into the default `App` / `PhaserGame` shell.

## Bumpkin maze (`pacman/`)

Pac-Man-style grid: **Bumpkin** player and chasers, **resource/icon** walls and pickups (`@sl-assets`), local state in `pacmanExampleStore.ts`.

**Try it:** import and render `<PacmanExample />` from `examples/pacman/PacmanExample` (e.g. on a dev route or behind a feature flag).

**Docs:** rules and asset table live in `docs/GAME_SPEC.md` (example section) and paths under `src/examples/pacman/`.
